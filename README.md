# Discord RPC Manager

**Разработчик:** [egorkadaa](https://github.com/egorkadaa)

Удобное десктопное приложение для управления Discord Rich Presence (RPC).  
Позволяет настраивать и отображать кастомную активность в Discord без единой строки кода.

---

## Возможности

- **Подключение к Discord** — введи Client ID своего приложения с Discord Developer Portal и нажми «Подключиться»
- **Редактор активности** — настройка всех полей Rich Presence:
  - Заголовок и описание (Details / State)
  - Большая и маленькая иконки с подписями
  - До 2 кнопок с кастомными ссылками
  - Таймер (сейчас / кастомное время)
  - Party Size (количество игроков)
- **Пресеты** — сохраняй готовые активности и применяй их в один клик
- **Автовосстановление** — после перезапуска приложения активность автоматически применяется при подключении
- **Автообновления** — при выходе новой версии приложение предложит скачать её через браузер
- **Трей** — сворачивается в системный трей, работает в фоне
- **Тёмная тема** — стильный современный интерфейс

---

## Как использовать

1. Скачай последний релиз из [Releases](https://github.com/egorkadaa/discord-rpc-manager/releases)
2. Установи и запусти
3. Перейди на [Discord Developer Portal](https://discord.com/developers/applications)
4. Создай приложение → скопируй **Client ID** (Application ID)
5. Вставь Client ID в приложение и нажми **«Подключиться»**
6. Настрой активность в редакторе и нажми **«Применить»**

> **Важно:** Для отображения кнопок в RCP необходимо указать хотя бы один Redirect URL в настройках OAuth2 твоего приложения в Discord Developer Portal.
